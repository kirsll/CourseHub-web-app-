КРАТКИЙ ПРИМЕР СТРУКТУРЫ ОСНОВНЫХ ТАБЛИЦ

Таблица users (пользователи)

  • user_id (PK)
  • email (UNIQUE)
  • password_hash
  • first_name
  • last_name
  • role (student/instructor/admin)
  • registration_date
  • is_active

Таблица courses (курсы)

  • course_id (PK)
  • title
  • description
  • price
  • instructor_id (FK -> users)
  • category_id (FK -> categories)
  • level (beginner/intermediate/advanced)
  • duration_hours
  • created_date
  • is_published

Таблица enrollments (зачисления)

  • enrollment_id (PK)
  • user_id (FK -> users)
  • course_id (FK -> courses)
  • enrollment_date
  • completion_date
  • progress_percentage
  • status (active/completed/cancelled)

Таблица orders (заказы)

  • order_id (PK)
  • user_id (FK -> users)
  • order_date
  • total_amount
  • status (pending/completed/cancelled)
